📦 Instalasi

ℹ️ Info WindUI adalah library UI modern untuk Roblox yang mudah digunakan dan sangat customizable.

Method 1: Loadstring (Recommended)

Basic Loadstring
local WindUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Footagesus/WindUI/main/dist/main.lua"))()

Method 2: Development Mode

Dengan Local Development Server
local WindUI
if game:GetService("RunService"):IsStudio() then
    WindUI = require(game:GetService("ReplicatedStorage"):WaitForChild("WindUI"):WaitForChild("Init"))
else
    local success, localBuild = pcall(function()
        return loadstring(game:HttpGet("http://localhost:8642/dist/main.lua"))()
    end)
    if success and localBuild then
        WindUI = localBuild
    else
        WindUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Footagesus/WindUI/main/dist/main.lua"))()
    end
end

if not WindUI then return end
💡 Tip Selalu cek apakah WindUI berhasil di-load sebelum melanjutkan dengan if not WindUI then return end

🚀 Quick Start

Berikut adalah contoh minimal untuk membuat UI dengan WindUI:

Script Minimal
-- Load WindUI
local WindUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/Footagesus/WindUI/main/dist/main.lua"))()

-- Buat Window
local Window = WindUI:CreateWindow({
    Title = "Script Saya",
    Icon = "rbxassetid://90566677928169",
    Folder = "MyScriptConfig"
})

-- Buat Tab
local Tab = Window:Tab({
    Title = "Main",
    Icon = "home"
})

-- Tambahkan Toggle
Tab:Toggle({
    Title = "Auto Farm",
    Value = false,
    Callback = function(value)
        print("Auto Farm:", value)
    end
})

-- Tambahkan Button
Tab:Button({
    Title = "Click Me",
    Callback = function()
        print("Button clicked!")
    end
})

🪟 Window

Window adalah container utama untuk semua elemen UI Anda.

Properti Window

Properti Type Default Deskripsi
Title string "UI Library" Judul window
Icon string nil Asset ID atau icon name
Author string nil Nama creator
Folder string nil Nama folder untuk config
Theme string nil Nama tema yang digunakan
SideBarWidth number 200 Lebar sidebar (pixels)
ToggleKey KeyCode nil Tombol untuk toggle UI
KeySystem table nil Config key system

Contoh Window

Window Basic
local Window = WindUI:CreateWindow({
    Title = "Script Saya",
    Icon = "rbxassetid://90566677928169",
    Author = "Nama Anda",
    Folder = "MyScriptConfig",
    Theme = "Arcvour",
    SideBarWidth = 180,
    ToggleKey = Enum.KeyCode.K
})
Window dengan Gradient Title
-- Fungsi gradient helper
local function gradient(text, startColor, endColor)
    local result = ""
    for i = 1, #text do
        local t = (i - 1) / math.max(#text - 1, 1)
        local r = math.floor((startColor.R + (endColor.R - startColor.R) * t) * 255)
        local g = math.floor((startColor.G + (endColor.G - startColor.G) * t) * 255)
        local b = math.floor((startColor.B + (endColor.B - startColor.B) * t) * 255)
        result = result .. "<font color=\"rgb(" .. r .. ", " .. g .. ", " .. b .. ")\">" .. text:sub(i, i) .. "</font>"
    end
    return result
end

local Window = WindUI:CreateWindow({
    Title = gradient("Premium HUB", Color3.fromHex("#8C46FF"), Color3.fromHex("#BE78FF")),
    Icon = "rbxassetid://90566677928169",
    Folder = "PremiumConfig",
    Theme = "Arcvour"
})
Window dengan Key System
local Window = WindUI:CreateWindow({

    }
})

📐 Layout & Structure

WindUI menggunakan struktur hierarkis: Window → Section → Tab → Section → Elements

Tab

Tab adalah container utama dalam sidebar untuk mengelompokkan fitur-fitur.

Membuat Tab
local PlayerTab = Window:Tab({
    Title = "Player",
    Icon = "user",
    IconColor = Color3.fromRGB(100, 150, 255),
    Border = true
})

Section

Section digunakan untuk mengelompokkan elemen dalam tab.

Section dengan Box
local MovementSection = PlayerTab:Section({
    Title = "Movement",
    Desc = "Pengaturan pergerakan karakter",
    Box = true,
    Opened = true
})

Group

Group untuk arrange elemen secara horizontal.

Membuat Group
local ButtonGroup = PlayerTab:Group()
ButtonGroup:Button({ Title = "Start", Icon = "play" })
ButtonGroup:Space()
ButtonGroup:Button({ Title = "Stop", Icon = "stop" })
ℹ️ Info Gunakan Section untuk organisasi vertikal dan Group untuk organisasi horizontal.

🔘 Button

Button untuk trigger actions/fungsi.

Properti

Properti Type Default Deskripsi
Title string "Button" Teks button
Desc string nil Deskripsi button
Icon string "mouse-pointer-click" Icon yang ditampilkan
Callback function function() end Fungsi yang dipanggil

Contoh

Button Basic
Tab:Button({
    Title = "Click Me",
    Callback = function()
        print("Button clicked!")
    end
})
Button dengan Description & Icon
Tab:Button({
    Title = "Teleport Spawn",
    Desc = "Teleport ke spawn point",
    Icon = "map-pin",
    Callback = function()
        local player = game.Players.LocalPlayer
        player.Character:SetPrimaryPartCFrame(CFrame.new(0, 50, 0))
        print("Teleported to spawn!")
    end
})
Button dengan Confirmation
Tab:Button({
    Title = "Reset Character",
    Desc = "Matikan karakter Anda",
    Icon = "refresh-cw",
    Callback = function()
        Window:Dialog({
            Title = "Konfirmasi",
            Content = "Yakin ingin reset karakter?",
            Buttons = {
                {
                    Title = "Ya",
                    Callback = function()
                        game.Players.LocalPlayer.Character.Humanoid.Health = 0
                    end
                },
                { Title = "Batal", Callback = function() end }
            }
        })
    end
})

Methods

local MyButton = Tab:Button({ Title = "My Button" })

-- Lock button (disable)
MyButton:Lock("Please wait...")

-- Unlock button (enable)
MyButton:Unlock()

-- Highlight button (blink effect)
MyButton:Highlight()

🎚️ Toggle

Toggle/Switch untuk boolean states. Tersedia dalam 2 mode: Toggle dan Checkbox.

Properti

Properti Type Default Deskripsi
Title string "Toggle" Teks toggle
Value boolean false Default state
Type string "Toggle" "Toggle" atau "Checkbox"
Flag string nil Flag untuk auto-save
Callback function function() end Callback dengan parameter bool

Contoh

Toggle Basic
Tab:Toggle({
    Title = "Auto Farm",
    Value = false,
    Callback = function(enabled)
        if enabled then
            print("Auto Farm: ON")
        else
            print("Auto Farm: OFF")
        end
    end
})
Toggle dengan Auto-Save
Tab:Toggle({
    Title = "Speed Hack",
    Desc = "Tingkatkan kecepatan jalan",
    Value = false,
    Flag = "SpeedHackEnabled", -- Akan auto-save!
    Callback = function(enabled)
        local player = game.Players.LocalPlayer
        if enabled then
            player.Character.Humanoid.WalkSpeed = 100
        else
            player.Character.Humanoid.WalkSpeed = 16
        end
    end
})
Checkbox Type
Tab:Toggle({
    Title = "Show FPS",
    Value = false,
    Type = "Checkbox",
    Flag = "ShowFPS",
    Callback = function(show)
        -- Toggle FPS display
    end
})
Master Toggle
local SubToggle1, SubToggle2

local MasterToggle = Tab:Toggle({
    Title = "Enable All",
    Value = false,
    Callback = function(enabled)
        SubToggle1:Set(enabled, true) -- With callback
        SubToggle2:Set(enabled, true)
    end
})

SubToggle1 = Tab:Toggle({ Title = "Feature 1" })
SubToggle2 = Tab:Toggle({ Title = "Feature 2" })

Methods

local MyToggle = Tab:Toggle({ Title = "My Toggle" })

-- Set value
MyToggle:Set(true) -- No callback
MyToggle:Set(true, true) -- With callback

-- Lock/Unlock
MyToggle:Lock("Feature disabled")
MyToggle:Unlock()
💡 Tip Gunakan Flag untuk auto-save state toggle. Gunakan Type = "Checkbox" untuk style checkbox.

🎚️ Slider

Slider untuk input nilai numerik dengan range min-max.

Slider Basic
Tab:Slider({
    Title = "Walk Speed",
    Value = { Min = 16, Max = 200, Default = 16 },
    Step = 1,
    Flag = "WalkSpeed",
    Callback = function(value)
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
    end
})
Slider dengan Icons
Tab:Slider({
    Title = "FOV",
    Value = { Min = 70, Max = 120, Default = 90 },
    Step = 1,
    Icons = {
        From = "minus",
        To = "plus"
    },
    Callback = function(fov)
        workspace.CurrentCamera.FieldOfView = fov
    end
})
ℹ️ Info Gunakan Step = 0.1 untuk decimal slider. Icons dari dan ke bisa custom sesuai kebutuhan.

💾 Config Manager

Sistem untuk menyimpan dan memuat konfigurasi user.

💡 Auto-Save Setiap elemen dengan properti Flag akan otomatis tersimpan dan dimuat!
Save & Load Config
-- Save config
Tab:Button({
    Title = "Save Config",
    Icon = "save",
    Callback = function()
        Window.ConfigManager:Save("MyConfig")
        Window:Notify({
            Title = "Saved",
            Content = "Config berhasil disimpan",
            Icon = "check"
        })
    end
})

-- Load config
Tab:Button({
    Title = "Load Config",
    Icon = "folder-open",
    Callback = function()
        Window.ConfigManager:Load("MyConfig")
        Window:Notify({
            Title = "Loaded",
            Content = "Config berhasil dimuat",
            Icon = "check"
        })
    end
})

✨ Best Practices

1. Gunakan Flag untuk Auto-Save

-- ✅ BAGUS
Tab:Toggle({
    Title = "Auto Farm",
    Flag = "AutoFarmEnabled", -- Akan auto-save!
    Callback = function(v) end
})

-- ❌ BURUK (tidak akan tersimpan)
Tab:Toggle({
    Title = "Auto Farm",
    Callback = function(v) end
})

2. Berikan Deskripsi yang Jelas

-- ✅ BAGUS
Tab:Toggle({
    Title = "Speed Hack",
    Desc = "Meningkatkan kecepatan. Jangan terlalu tinggi!",
    Callback = function(v) end
})

3. Error Handling yang Baik

-- ✅ BAGUS: Dengan error handling
Tab:Button({
    Title = "Teleport",
    Callback = function()
        local success, err = pcall(function()
            -- Teleport logic
        end)
        
        if success then
            Window:Notify({ Title = "Success", Icon = "check" })
        else
            Window:Notify({ Title = "Error", Content = tostring(err), Icon = "x" })
        end
    end
})

4. Organisasi yang Baik

-- ✅ BAGUS: Terorganisir dengan Section
local MovementSection = Tab:Section({ Title = "Movement", Box = true })
MovementSection:Slider({ Title = "Walk Speed" })
MovementSection:Slider({ Title = "Jump Power" })

local CombatSection = Tab:Section({ Title = "Combat", Box = true })
CombatSection:Toggle({ Title = "Auto Attack" })
